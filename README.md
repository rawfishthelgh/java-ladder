## 기능 정의

- Name
  - [x] name의 길이가 1~5글자임을 검증할 수 있다
  - [x] 자신의 이름을 답할 수 있다
- Block
  - [x] 본인이 다음 Block으로 가는 경로가 있는지를 판단할 수 있다(pass)
- strategy
  - [x] 다음 사다리로 이동하는 경로를 생성한다
- 예외처리
  - [x] 사다리의 높이는 1이상이어야 한다
  - [x] 구분자가 포함되어야 한다
